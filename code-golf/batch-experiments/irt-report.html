关键结论： RMSEr 和 IRT 区分度（a）几乎无关（Spearman ρ = 0.15）。
RMSEr 测量的是「分数有多分散」，IRT a 测量的是「强弱队是否按分数正确排序」。
高 RMSEr 但低 a 的题 = 噪音大但无区分力，应剔除。
低 RMSEr 但高 a 的题 = 分数接近但排序精准，是黄金题目。
