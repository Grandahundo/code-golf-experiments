洞察：AI 和人类高度一致（Spearman ρ=0.83，p<0.001）。对人类难的任务对 AI 也难。 但差距随难度呈非线性增长——简单题 1.18x，中等题 1.90x，困难题 2.57x。 这不是因为困难题的最优解更高（其实差不多，95B vs 69B），而是因为困难题的方差大： 顶尖人类能找到好解，普通人类找不到，AI 更接近普通人类水平。

洞察：DS V4 Pro 排在全球 48 支顶尖队伍的第 27 名——恰好是中位数。 它之上的每一支队伍都由人类专家组成，每人花了数小时优化每个任务。 而 DS 只用了 30 轮自动化对话，每轮数秒到数十秒。 这意味着：当前前沿模型在 code golf 上已经达到熟练人类的竞争力， 根本的瓶颈不是"能不能做对"，而是"能不能压得足够短"。

洞察：最强预测因子不是 IRT 区分度（a），而是人类技能方差（Q50/Best）。 这意味着 AI 的弱点和人类的"能力梯度"是不同维度的——IRT a 区分的是顶尖 vs 普通人类， 但 AI 被困在所有人都能找到基础解但只有少数人能极端优化的任务上。

Prompt 工程：下一步方向

当前实验使用通用 code golf prompt。模板 agent_prompt_template.md 提供了更结构化的方案：

  • 显式 submit_code 工具语义：让模型明确知道每轮的"动作"和"回报"结构
  • 四步策略：理解规则 → AC 优先 → 增量压缩 → 死磕换思路
  • 结构化观测模板：AC 新纪录 / AC 未破纪录 / WA 含失败样例 / RE 含 traceback / TLE
  • Golf tricks 清单：lambda、comprehension、zip 转置、walrus 等显式列出

这是明确的下一步实验方向——预期在压缩比上会有提升，尤其在中等和困难题上。
