⛳ Code Golf Benchmark

Agentic Loop Architecture — spec v2

Inputs

Tasks (400)

taskNNN.json — Google Code Golf 2025
ARC-style grid transformation puzzles.
Each has train (2-10 examples),
test + arc-gen (hidden judge set).

Generators

gen.py — ground-truth transformation
Available for task001–010 only.
Injected directly into model prompt.

Config

config.yaml
Models, concurrency, timeouts,
token budgets, seed count.

Pool Records

records.json
Rolling best-bytes-per-task across
all models & seeds. Updated by scoring.

🧠 Agentic Loop — run_job()

1
Build Prompt
prompts.py → system + gen.py + train examples + state + scratchpad + history
2
Call LLM
llm.py → OpenAI-compatible / Anthropic, retry 5× with backoff
3
Parse Reply
Extract last ```python block + <notes> → code & scratchpad
4
Judge
executor.py → compile check → subprocess → judge_wrapper.py
5
Record & Update
Append trajectory.jsonl, update best.py if new record, write meta.json

📋 Judge Wrapper (subprocess)

import user code → set resource limits → signal.alarm per-case → call p(input) → normalize (JSON roundtrip, true→1, false→0) → numpy.array_equal → stream JSON-lines results → short-circuit after 3 failures

🔍 7-level error taxonomy

parse_error | syntax_error | no_p | crash | timeout | wrong | correct
+ api_error (not counted as a round)

Outputs

📁 runs/{model}/{task}/seed{n}/

├── trajectory.jsonl  per-round records
├── best.py  current best solution
└── meta.json  job summary + stats

Scoring

scoring.py
Reads all meta.json → pool records →
score = record_bytes / your_bytes.
Best-of-3 & median-of-3 per model.

📊 Leaderboard Outputs

├── leaderboard.csv  per-model summary
├── per_task.csv  per-task detail
└── records.json  pool records

Runner

runner.py — async scheduler
job matrix = tasks × models × seeds
per-model concurrency via semaphore
Ctrl-C graceful shutdown
--dry-run cost estimate
Resume from meta.json

Data Flow (one round)

task.json → gen.py → task_loader.py
→ TaskBundle → prompts.py
→ build_user_prompt() → llm.py
→ chat() → executor.py
→ judge() → trajectory.jsonl
+ best.py
🔄 Loop: rounds 1→30 per (task, model, seed)
⚡ Concurrency: per-model semaphore (default 8)
💾 Resume: reads trajectory.jsonl, skips finished meta.json
💰 Budget: 1.5M tokens per job, 22K tokens per prompt
